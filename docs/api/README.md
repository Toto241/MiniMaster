# API Documentation

This directory contains OpenAPI specifications and API documentation for the Mini-Master project.

## Structure

- `cloud-functions-api.yaml` - OpenAPI 3.0 specification for Cloud Functions

## Usage

API documentation can be viewed using Swagger UI or any OpenAPI-compatible tool.

## Maintenance

API documentation must be updated whenever:
- New endpoints are added
- Existing endpoints are modified
- Request/response schemas change
- Authentication requirements change

Refer to CONTRIBUTING.md for detailed documentation requirements.
