# Country Reviews (Target Markets)

Each file in this folder is the legal-operational review artifact for one target market.

Minimum release rule for every market:

1. Google Play baseline passed.
2. Local legal review signed off.
3. Terms/Privacy versioned and deployed for country+locale.
4. Re-consent trigger defined for major legal or product changes.
