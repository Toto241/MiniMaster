# MiniMaster Desktop Launcher

Native Desktop-Oberfläche (Electron) für die vorhandenen PC-Panels.

## Start

Vom Repository-Root:

```bash
npx electron desktop/main.js
```

## Enthaltene Oberflächen

- Eltern-Panel: `web-control/index.html`
- Operator/Admin-Panel: `admin-panel/index.html`
- Audit-Log-Ansicht: `admin-panel/logs.html`

## Hinweis

Die Panels benötigen eine konfigurierte Firebase-Verbindung (siehe jeweilige README/Config-Dateien).
