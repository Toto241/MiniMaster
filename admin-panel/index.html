Willkommen beim MiniMaster Operator Dashboard

Dieses Dashboard ist ausschließlich für den Betreiber von MiniMaster vorgesehen – nicht für Eltern oder Kinder. Folgen Sie den Schritten unten, um Ihr System einzurichten und Zugang zu erhalten.

Freigabe-Status und Commissioning-Nachweise können direkt aus dem QA-Register abgeleitet werden. Bereits abgeschlossene Qualitätssicherungsbereiche wie Play-Store-Readiness, Android-/Support-/Compliance-Commissioning und AI-Runtime werden ausschließlich im Bereich Qualitätssicherung gepflegt. Alle Daten sind lokal persistiert und im Admin-Panel einsehbar. CLI-Befehle können direkt aus diesem Dashboard ausgeführt werden.

Schritt 1: Firebase-Projekt verbinden

Damit dieses Dashboard mit Ihrem MiniMaster-Backend kommunizieren kann, benötigt es die Firebase-Webkonfiguration Ihres Projekts. Sie finden diese Werte in der Firebase Console unter Projekteinstellungen → Allgemein → Ihre Apps → Web-App.

Konfiguration aus dem Internet laden

Optional: Lade eine veröffentlichte JSON-/JS-Datei (z. B. aus Hosting oder GitHub Raw), die ein firebaseConfig-Objekt enthält.

Live-Sync ist aktiv: Bei jeder Eingabe werden gültige Werte automatisch lokal gespeichert und direkt als aktive Firebase-Einstellung übernommen.

Diese Felder werden – sobald gefüllt – beim Klick auf Übertragen in die Datei .env im Projektstamm geschrieben. Felder leer lassen heißt: nichts ändern. Geheime Werte werden im Eingabefeld nicht zurückgespiegelt – beim Laden zeigen wir nur eine Maske.

Konfigurations-Snapshots (Schutz vor Datenverlust)

Vor jeder Uebertragung und vor Git-Operationen (checkout/merge/rebase) wird automatisch ein Snapshot der Konfig-Pflicht-Dateien ausserhalb des Repos abgelegt. Per Wiederherstellen kann ein frueherer Stand zurueckgeholt werden – der aktuelle Stand wird vorher automatisch als pre-restore-Snapshot gesichert.

Wo finde ich diese Werte?
  1. Öffnen Sie die Firebase Console
  2. Wählen Sie Ihr MiniMaster-Projekt
  3. Klicken Sie auf das Zahnrad-Symbol → Projekteinstellungen
  4. Scrollen Sie zu Ihre Apps → wählen Sie die Web-App (</>)
  5. Falls noch keine Web-App existiert: App hinzufügen → Web
  6. Kopieren Sie die Werte aus dem firebaseConfig-Objekt in die Felder oben
